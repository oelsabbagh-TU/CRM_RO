# cross_border-CM-analysis RO settlement

This folder is the self-contained analysis workspace for the thesis RO extension.

## What is here
- `RO_settlement.jl`: settlement post-processing on the saved equilibrium outputs
- `CM_analysis.ipynb`: capacity-market summary and figures
- `RO_analysis.ipynb`: RO settlement summary and figures
- `Results/`: analysis outputs used by the notebooks and settlement script

## Conventions
- all paths are local to `cross_border-CM-analysis`
- the RO settlement output is written inside `RO_Settlement/Results/`
- the analysis stays in this folder and does not depend on the main repo
